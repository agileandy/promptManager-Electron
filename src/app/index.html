📝 AI Prompt Manager Help

➕ Creating Prompts: Click "New Prompt" to create a new AI prompt with title, description, and text.

🔍 Search: Use the search bar to find prompts by title, description, or content.

📊 Sorting: Use the dropdown to sort by creation date, title, usage count, or last used date.

📋 Copy Prompts: Click the copy icon to copy prompt text to clipboard and track usage.

✏️ Edit Prompts: Click the edit icon to create a new version while preserving history.

📥 Import/Export: Use Import/Export buttons to backup or share your prompts as JSON files.

🌓 Theme Toggle: Switch between light and dark modes with the theme button.

📖 Version History: Click the history icon to view all versions of a prompt.
