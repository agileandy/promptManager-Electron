📝 AI Prompt Manager Help

Welcome to AI Prompt Manager! 🎉

Your personal AI prompt library for organizing, managing, and enhancing your AI interactions.

🎯 Quick Start

  • 1. Click New Prompt to create your first prompt
  • 2. Add a title, description, and your prompt text
  • 3. Use tags to organize your prompts
  • 4. Copy prompts to use with your favorite AI tools

Generate and optimize prompts with AI

📊 Usage Analytics

Track which prompts you use most

Managing Your Prompts 📝

➕ Creating Prompts

  • • Click "New Prompt" button in the top toolbar
  • • Fill in the title, description, and prompt text
  • • Add relevant tags for organization
  • • Use the 🤖 AI Generate button to create descriptions automatically

✏️ Editing Prompts

  • • Click the edit icon (✏️) on any prompt card
  • • Editing creates a new version while preserving history
  • • Use ✨ AI Optimize to improve your prompt text
  • • All changes are automatically saved

📋 Using Prompts

  • • Click the copy icon (📋) to copy prompt text to clipboard
  • • Usage count automatically increments when copied
  • • Last used date is updated for better sorting

📖 Version History

  • • Click the history icon (📖) to view all versions
  • • Compare different versions side by side
  • • Restore previous versions if needed

🔍 Search & Sort

  • • Use the search bar to find prompts by title, description, or content
  • • Sort by: Creation Date, Title, Usage Count, or Last Used
  • • Filter by tags using the tag sidebar

Tags & Organization 🏷️

Keep your prompts organized with our powerful tagging system.

🎯 Creating Tags

  • • Tags are created automatically when you type them in the tag field
  • • Use descriptive names like "writing", "coding", "marketing"
  • • Tags are case-insensitive and automatically formatted

🖱️ Drag & Drop Tags

✨ New Feature!

  • • Drag tags from the sidebar directly onto prompt cards
  • • Visual feedback shows when you can drop a tag
  • • Instant organization - no need to edit each prompt
  • • Bulk tagging - quickly organize multiple prompts

📊 Tag Management

  • • View all tags in the left sidebar with usage counts
  • • Click any tag to filter prompts by that tag
  • • Tags show the number of prompts they're applied to
  • • Unused tags are automatically cleaned up

AI-Powered Features 🤖

Enhance your prompts with built-in AI assistance.

🤖 AI Generation

  • • Auto-generate descriptions from your prompt text
  • • Click the 🤖 Generate button next to the description field
  • • AI analyzes your prompt and creates a helpful description
  • • Perfect for quickly documenting what your prompts do

✨ AI Optimization

  • • Improve your prompt text for better AI responses
  • • Click the ✨ Optimize button next to the prompt field
  • • AI suggests improvements for clarity and effectiveness
  • • Great for refining prompts that aren't giving good results

⚙️ AI Configuration

  • • Click the AI Settings button to configure providers
  • • Supports OpenRouter and Ollama providers
  • • Set your preferred default provider
  • • Configure API keys, models, and other settings

🔧 Provider Setup

OpenRouter Setup
  • 1. Get API key from openrouter.ai
  • 2. Enter API key in AI Settings → OpenRouter tab
  • 3. Choose your preferred model (free models available)
  • 4. Use the refresh button (🔄) to load available models
Ollama Setup
  • 1. Install Ollama locally from ollama.ai
  • 2. Start Ollama service (usually runs on localhost:11434)
  • 3. Configure endpoint in AI Settings → Ollama tab
  • 4. Models are automatically detected from your Ollama installation

Advanced Features ⚙️

📥 Import/Export

  • • Export: Backup all your prompts as a JSON file
  • • Import: Restore prompts from a backup file
  • • Share: Exchange prompt collections with others
  • • Files include prompts, tags, and usage statistics

🗄️ Database Management

  • • Click "View Database" to see raw data structure
  • • All data is stored locally in your browser
  • • No data is sent to external servers (except AI providers)
  • • Regular backups recommended via Export function

🌓 Customization

  • • Theme Toggle: Switch between light and dark modes
  • • Sorting Options: Customize how prompts are displayed
  • • AI Settings: Configure temperature, timeouts, and retries
  • • System Prompts: Customize AI behavior for generation and optimization

🔒 Privacy & Security

  • • All prompts stored locally in your browser
  • • AI features only send data when you explicitly use them
  • • API keys are stored securely in local storage
  • • No telemetry or usage tracking
